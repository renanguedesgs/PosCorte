## Projetos

| Projeto | Responsabilidade |
|---------|-----------------|
| `PosCorte.API` | Controllers, Services, Middlewares, DI |
| `PosCorte.Domain` | Entities, ValueObjects |
| `PosCorte.Tests` | Testes unitários com xUnit + Moq |

## Padrões Utilizados

- **Repository Pattern** com interface genérica `IRepositorio<T>`
- **Dependency Injection** nativo do ASP.NET Core
- **Markup Inverso** para precificação
- **Escrow** para segurança financeira
